# JSX Preview

Preview JSX/TSX elements visually in VS Code without leaving the editor.

## Features

- Hover over any JSX element → click **[JSX Preview]** → see a visual preview in a side panel
- Supports HTML elements (`<div>`, `<button>`, etc.) and custom components
- Tailwind CSS support with custom config (`tailwind.config.js`)
- Auto-updates on file save
- Works without running a dev server

## How it works

1. Hover over a JSX/TSX element
2. Click the `[JSX Preview]` link
3. A side panel opens with the rendered preview

## Requirements

- React and ReactDOM installed in your project
- Tailwind CSS installed for Tailwind support (optional)
