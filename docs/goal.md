# JSX Preview

## 課題
コードの見た目を確認するためにブラウザに切り替えるのが面倒。

## ゴール
JSX要素にホバー → [Preview] クリック → サイドパネルにプレビュー表示。
アプリ起動なしで動く。

## 現状（完了済み）
- ホバーで[Preview]リンク表示
- クリックでサイドパネルにHTML描画
- Tailwind CSS対応（CDN）

## 改善案
- ホバーのみでプレビュー表示（クリック不要）
- 表示方法をオプションで切り替え可能にする（ホバー / クリック / パネル等）
- styled-components / Emotion 対応
- Tailwind CDN → ローカルバンドル化（オフライン対応）
- JSX解析精度の向上（正規表現 → パーサー）
- カスタムコンポーネントの解決・描画
